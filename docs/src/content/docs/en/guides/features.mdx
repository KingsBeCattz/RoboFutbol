Our guide adds support for various gamepads, [click here to check which ones](https://bluepad32.readthedocs.io/en/latest/supported_gamepads/), and to various support for various bridges h.

At the same time it has several ways to control it, you can modulate the speed with triggers and shortcuts next to the bumpers (or R1/L1) and turn with the X axis of the left stick; modulate the speed with the Y axis of the left stick, control the turn with the X axis of the right stick; even a “universal” control with the d-pad.

<Aside type="caution">
This project was built using the ESP32 microcontroller, if you don't have experience using it we recommend you to watch the intro tutorials.
</Aside>
