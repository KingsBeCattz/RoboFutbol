## Special Modes

<Card title="Bluepad32 & PS2X - Special Modes">
  <Card title="Explanation" icon="magnifier">
    The special modes below apply to both Bluepad32 (ESP32) and PS2X (Arduino) controllers. Button names are shown for both platforms when they differ.

    - **Manual Drive — A (Xbox) / Cross (PS2):** Manual driving mode. Controls behave as standard: LT/LB control backward, RT/RB control forward; pressing LB or RB acts as a full-direction shortcut.

    - **Tank Drive — X (Xbox) / Square (PS2):** Tank driving mode. Uses same input mapping but increases turn sensitivity and keeps turning behavior tied to the same axis.

    - **Exposition Mode — Y (Xbox) / Triangle (PS2):** Exposition mode. Speed controls (triggers or axis) control both forward/backward speed and rotation: `255` = full right rotation, `-255` = full left rotation. Rotation behaves like tank mode.

    - **Controller Disconnect (Bluepad32 only):** Hold the system button (Xbox button) for **~2 seconds** to disconnect the controller from the ESP32.
  </Card>
</Card>

<Aside type="caution">
- Toggle selections (Select/Start) persist while the board remains powered; a controller disconnect does not reset them — only an board restart clears toggles.
- When multiple opposing speed inputs are active (e.g., full reverse and full forward), they cancel out resulting in stop; partial values combine algebraically.
</Aside>
