## Modos Especiales

<Card title="Bluepad32 & PS2X - Modos Especiales">
  <Card title="Explicación" icon="magnifier">
    Los siguientes modos especiales aplican tanto a los controladores Bluepad32 (ESP32) como a los PS2X (Arduino). Los nombres de botones se muestran para ambas plataformas cuando difieren.

    - **Manual Drive — A (Xbox) / Cruz (PS2):** Modo de conducción manual. Los controles se comportan de forma estándar: LT/LB controlan reversa, RT/RB controlan avance; presionar LB o RB actúa como atajo de dirección completa.

    - **Tank Drive — X (Xbox) / Cuadro (PS2):** Modo de conducción tipo tanque. Usa el mismo mapeo de entradas pero aumenta la sensibilidad de giro y mantiene el comportamiento de giro vinculado al mismo eje.

    - **Modo Exposición — Y (Xbox) / Triángulo (PS2):** Modo de exposición. Los controles de velocidad (triggers o ejes) controlan tanto la velocidad hacia adelante/atrás como la rotación: `255` = rotación completa a la derecha, `-255` = rotación completa a la izquierda. El giro se comporta igual que en modo tanque.

    - **Desconexión del Controller (solo Bluepad32):** Mantén presionado el botón del sistema (botón Xbox) por **~2 segundos** para desconectar el controller del ESP32.
  </Card>
</Card>

<Aside type="caution">
- Las selecciones conmutadas (Select/Start) persisten mientras la board permanezca encendida; una desconexión del controller no las reinicia — solo un reinicio de la board las limpia.
- Cuando múltiples entradas de velocidad opuestas están activas (por ejemplo, reversa total y avance total), se cancelan entre sí, resultando en parada; los valores parciales se combinan algebraicamente.
</Aside>
