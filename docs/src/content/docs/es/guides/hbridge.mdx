Nuestro código, además de soportar varios gamepads gracias a Bluepad32, también es compatible con distintos tipos de **puentes H (H-bridges)** gracias a su pinout.  
Los ejemplos mostrados aquí aplican tanto para **ESP32 (Bluepad32)** como para **boards compatibles con Arduino (PS2)**.

<Aside type="caution">
Este proyecto fue desarrollado utilizando **microcontroladores ESP32 y boards compatibles con Arduino**.  
Si no tienes experiencia con ellos, te recomendamos ver algunos tutoriales introductorios antes de continuar.
</Aside>

Como puedes ver, todos estos ejemplos tienen algo en común: la presencia de **IN1/IN3 e IN2/IN4**, que controlan la dirección del motor.  
En nuestro código hay soporte para un máximo de **dos motores** y **dos interfaces**:

- **Interfaz clásica** — salida simple HIGH o LOW  
- **Interfaz PWM** — salida PWM de 0 a 255
