## Flags de Configuración

La configuración se divide en dos grupos:  
los pines que controlan la **dirección del motor** y los pines que **habilitan el puente H** de cada motor.  
Solo **una opción por grupo** debe estar activa a la vez.

### Pines de dirección del motor

```cpp
// #define USE_PWM_MOTOR_PIN      // Usa pines PWM para dirección del motor
// #define USE_DIGITAL_MOTOR_PIN  // Usa pines digitales para dirección del motor
```

### Pines de habilitación (ENABLE)

```cpp
// #define USE_PWM_ENABLE_PIN     // Usa pines PWM para habilitar el motor
// #define USE_DIGITAL_ENABLE_PIN // Usa pines digitales para habilitar el motor
```

<Aside type="note">
Descomenta solo la opción que necesites para cada grupo.  
Por ejemplo, si quieres dirección PWM y habilitación digital, descomenta:

```cpp
#define USE_PWM_MOTOR_PIN
#define USE_DIGITAL_ENABLE_PIN
```
</Aside>

| ENABLE DIGITAL             | GPIO 32, 23           | 10, 11                  | Pines digitales dedicados como ENABLE                      |
