Nuestra guía añade soporte para varios gamepads, [haz clic aquí para comprobar cuáles](https://bluepad32.readthedocs.io/en/latest/supported_gamepads/), y para varios soporte para varios puentes h.

Al mismo tiempo tiene varias formas de controlarlo, puedes modular la velocidad con los gatillos y atajos junto a los parachoques (o R1/L1) y girar con el eje X del stick izquierdo; modular la velocidad con el eje Y del stick izquierdo, controlar el giro con el eje X del stick derecho; incluso un control «universal» con el d-pad.

<Aside type="caution">
Este proyecto fue construido usando el microcontrolador ESP32, si no tienes experiencia usándolo te recomendamos ver los tutoriales de introducción.
</Aside>
