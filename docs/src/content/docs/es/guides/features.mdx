Nuestra guía ofrece soporte para múltiples gamepads — [haz clic aquí para ver la lista de controladores compatibles](https://bluepad32.readthedocs.io/en/latest/supported_gamepads/) — así como compatibilidad con varios tipos de puentes H (*H-bridges*).

El sistema soporta tanto **ESP32 con controladores Bluetooth** como **boards compatibles con Arduino con controladores PS2**, ofreciendo múltiples modos de control configurables en tiempo de ejecución:

- **ESP32 (Bluepad32)**: la velocidad se modula usando los **triggers** (RT/LT), y el giro se controla con el **eje X del stick izquierdo** por defecto.
- **Arduino (PS2)**: el joystick izquierdo → avanzar/retroceder (eje Y), el joystick derecho → izquierda/derecha (eje X).  
- Los botones controlan los modos o el cambio de fuente de potencia:
  - **SELECT** → cambia la fuente de potencia
  - **START** → cambia la fuente de dirección
  - **X en Xbox / Cuadro en PlayStation / Y en Nintendo** → conducción manual
  - **Y en Xbox / Triángulo en PlayStation / X en Nintendo** → modo exposición
  - **A en Xbox / Cruz en PlayStation / B en Nintendo** → modo tanque
- También puedes usar el **d-pad** como controlador direccional universal.

<Aside type="caution">
Este proyecto está diseñado para ejecutarse en **microcontroladores ESP32 y boards compatibles con Arduino**.  
Si no estás familiarizado con ellos, te recomendamos revisar algunos tutoriales introductorios antes de continuar.
</Aside>
